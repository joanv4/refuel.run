Privacidad y Protección de Datos

¿Qué datos recopilamos?

Recopilamos únicamente los datos necesarios para calcular el plan nutricional:

  • Peso, altura, edad y sexo
  • Experiencia en resistencia
  • Características del evento (distancia, tiempo, desnivel, altitud)
  • Condiciones ambientales (temperatura, clima)

Estos datos se procesan localmente en tu navegador y no se almacenan en servidores.

¿Cómo se usan los datos?

Los datos se utilizan exclusivamente para:

  • Calcular el gasto energético metabólico
  • Determinar necesidades de carbohidratos y electrolitos
  • Generar recomendaciones nutricionales
  • Ajustar según condiciones ambientales

Todo el procesamiento se realiza en tu dispositivo, sin envío de datos a servidores externos.

¿Usamos cookies?

No utilizamos cookies para rastreo o análisis de usuarios.

Únicamente usamos:

  • Cookies técnicas necesarias para el funcionamiento del sitio
  • Almacenamiento local del navegador para mantener el estado del formulario
  • Datos temporales que se eliminan al cerrar el navegador

No hay seguimiento de comportamiento ni datos compartidos con terceros.

¿Dónde se almacenan los datos?

Todos los datos se procesan y almacenan localmente en tu navegador:

  • No se envían datos a servidores externos
  • No hay base de datos ni almacenamiento en la nube
  • Los datos se eliminan automáticamente al cerrar el navegador
  • No hay registro de IP ni información de identificación

El sitio funciona completamente offline una vez cargado.

¿Cómo protegemos tus datos?

Implementamos múltiples capas de protección enfocadas en la privacidad:

  • Procesamiento local sin envío de datos: Todo el cálculo se realiza en tu navegador, sin transmisión de datos personales
  • Conexión HTTPS encriptada: Todas las conexiones externas usan encriptación SSL/TLS
  • No hay formularios que envíen datos: El formulario no tiene action ni method que envíe información a servidores
  • Código fuente disponible: El código es transparente y puede ser auditado por cualquier persona
  • Dependencias externas mínimas: Solo Google Fonts para la interfaz visual (no envía datos personales)

Tu privacidad está protegida por diseño desde el principio, con énfasis en el procesamiento local y la transparencia.

¿Compartimos datos con terceros?

No compartimos datos con terceros bajo ninguna circunstancia:

  • No hay integración con servicios externos
  • No hay análisis de uso ni métricas
  • No hay publicidad ni marketing
  • No hay venta de datos

Los datos permanecen exclusivamente en tu dispositivo.

¿Cuáles son mis derechos?

Tienes derecho a:

  • Acceder a toda la información sobre tus datos
  • Solicitar eliminación de datos (aunque no se almacenan)
  • Conocer cómo se procesan los datos
  • Revocar consentimiento en cualquier momento
  • Contactar para cualquier consulta sobre privacidad

Como no almacenamos datos, la mayoría de estos derechos se cumplen automáticamente.
